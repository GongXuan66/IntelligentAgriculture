# 3 智能灌溉模块

## 功能

- 土壤湿度检测
- 自动灌溉
- 手动灌溉

------

## 数据结构

灌溉记录：

```

```

属性：

```
startTime
endTime

```

------

## 自动灌溉算法设计

系统根据 **土壤湿度阈值**判断是否灌溉。

算法逻辑：

```
读取土壤湿度

if 湿度 < 设定阈值
    启动水泵
    灌溉指定时间
    关闭水泵
记录灌溉数据
```

示例：

```
阈值 = 40%

当前湿度 = 30%

启动灌溉
```

------

## 核心方法

检测湿度：

```
checkSoilMoisture()
```

启动灌溉：

```
startIrrigation()
```

停止灌溉：

```
stopIrrigation()
```

记录灌溉数据：

```
recordIrrigation()
```

# 四、通信模块设计

APP需要与网关通信。

通信方式：

```
HTTP 或 MQTT
```

------

## 数据接收流程

```
网关发送传感器数据
        ↓
APP接收数据
        ↓
解析JSON数据
        ↓
更新UI
```

示例数据：

```
{
 "temperature":26,
 "humidity":65,
 "light":320,
 "co2":450
}
```

------

## 控制指令流程

```
用户点击按钮
        ↓
APP发送控制指令
        ↓
网关接收指令
        ↓
设备执行操作
```

示例：

```
{
 "device":"waterPump",
 "command":"on"
}
```
