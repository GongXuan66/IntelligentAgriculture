# 一、从赛题中提取的软件开发核心任务

你负责的软件部分，本质是开发一个 **基于 OpenHarmony 的智慧农业控制 APP + 网关通信逻辑**。

核心任务可以提炼为 **5 个软件模块**：

### 1 智能控制中心（设备管理模块）

功能：

- 设备连接管理
- 设备状态管理
- 指令发送
- 指令转发
- 联动规则配置

本质功能：

APP ⇄ 网关 ⇄ 设备通信控制

例如：

APP 发送：

```
开启水泵
```

网关转发：

```
waterPump = ON
```

设备执行。

------

### 2 智能灌溉模块

需要开发的功能：

1. 土壤温湿度数据采集
2. APP 实时展示
3. 自动计算灌溉水量算法
4. 手动灌溉控制
5. 灌溉状态反馈

核心逻辑：

```
读取湿度
↓
判断是否低于阈值
↓
计算需要灌溉水量
↓
启动水泵
↓
更新状态
```

------

### 3 智能光照模块

功能：

1. 光照强度采集
2. APP 展示光照数据
3. LED 补光控制
4. LED 亮度调节

控制逻辑示例：

```
光照 < 300lux
↓
自动开启LED
↓
调整亮度
```

------

### 4 环境监测模块

采集：

- 温度
- 湿度
- 光照
- 二氧化碳

APP 展示：

- 实时数据
- 图表展示

功能：

- 阈值配置
- 自动报警

例如：

```
温度 > 35℃
↓
发送预警
```

------

### 5 扩展设备模块

需要接入 **1 种额外设备**

例如：

| 设备       | 功能       |
| ---------- | ---------- |
| 风扇       | 通风降温   |
| 摄像头     | 农田监控   |
| PH 传感器   | 土壤酸碱度 |
| 气体传感器 | 有害气体   |

扩展目标：

验证系统 **可扩展性**。
