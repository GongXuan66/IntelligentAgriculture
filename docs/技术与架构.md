## 1. 项目概述

### 1.1 项目背景

本项目是一个面向智慧农业的综合性管理平台，整合物联网传感、智能灌溉、环境监测、设备控制、AI辅助决策等功能，实现农业生产的数字化、智能化管理。

### 1.2 系统目标

- **精准灌溉**: 基于多维数据的智能灌溉决策，实现节水增产
- **智能监控**: 实时环境监测与异常预警
- **设备联动**: 自动化设备控制与远程管理
- **AI辅助**: 自然语言交互的智能助手

### 1.3 系统范围

```
┌─────────────────────────────────────────────────────────────┐
│                     智慧农业管理系统                          │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐           │
│  │  环境监测   │ │  智能灌溉   │ │  设备控制   │           │
│  └─────────────┘ └─────────────┘ └─────────────┘           │
│  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐           │
```

### 5.2 核心表结构

#### 5.2.1 用户表 (user)

```sql
CREATE TABLE user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    nickname VARCHAR(50),
    phone VARCHAR(20),
    email VARCHAR(100),
    role VARCHAR(20) DEFAULT 'user',
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 5.2.2 农场表 (farm)

```sql
CREATE TABLE farm (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    location VARCHAR(255),
    area DECIMAL(10,2),
    description TEXT,
    owner_id BIGINT,
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (owner_id) REFERENCES user(id)
);
```

#### 5.2.3 监控点表 (monitor_point)

```sql
CREATE TABLE monitor_point (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    farm_id BIGINT NOT NULL,
    location VARCHAR(255),
    latitude DECIMAL(10,7),
    longitude DECIMAL(10,7),
    soil_type VARCHAR(50),
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (farm_id) REFERENCES farm(id)
);
```

#### 5.2.4 传感器数据表 (sensor_data)

```sql
CREATE TABLE sensor_data (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    monitor_point_id BIGINT NOT NULL,
    temperature DECIMAL(5,2),
    humidity DECIMAL(5,2),
    soil_moisture DECIMAL(5,2),
    light_intensity INT,
    co2_concentration INT,
    ph_value DECIMAL(4,2),
    ec_value DECIMAL(6,2),
    recorded_at DATETIME NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (monitor_point_id) REFERENCES monitor_point(id),
    INDEX idx_recorded_at (recorded_at),
    INDEX idx_point_time (monitor_point_id, recorded_at)
);
```

#### 5.2.5 灌溉记录表 (irrigation_log) ⭐创新设计

```sql
CREATE TABLE irrigation_log (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    monitor_point_id BIGINT NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME,
    water_amount DECIMAL(10,2),
    duration INT,
    mode TINYINT DEFAULT 0 COMMENT '0自动 1手动',
    
    -- 决策数据 (创新点：合并决策信息)
    decision_type VARCHAR(30) COMMENT 'predictive/adaptive/stage_based/manual',
    current_moisture DECIMAL(5,2),
    predicted_moisture DECIMAL(5,2),
    crop_stage VARCHAR(30),
    irrigation_factor DECIMAL(5,2),
    confidence DECIMAL(5,2),
    
    -- 效果追踪
    soil_moisture_before DECIMAL(5,2),
```

### 7.2 EWMA湿度预测算法

**算法原理**: 指数加权移动平均(Exponentially Weighted Moving Average)

**核心公式**:
```
S(t) = α × X(t) + (1-α) × S(t-1)

其中:
- S(t): t时刻的平滑值
- X(t): t时刻的实际观测值
- α: 平滑系数 (0 < α < 1)
```

**趋势预测**:
```java
// MoisturePredictor.java 核心逻辑
public MoisturePrediction predict(Long pointId, int hoursAhead) {
    // 1. 获取历史数据
    List<Double> history = getMoistureHistory(pointId, 24);
    
    // 2. 计算EWMA平滑序列
    double alpha = 0.3;
    List<Double> smoothed = calculateEWMA(history, alpha);
    
    // 3. 估计趋势
    double trend = estimateTrend(smoothed);
    
    // 4. 外推预测
    double lastValue = smoothed.get(smoothed.size() - 1);
    double prediction = lastValue + trend * hoursAhead;
    
    // 5. 计算置信度
    double confidence = calculateConfidence(history, smoothed);
    
    return new MoisturePrediction(prediction, confidence, trend);
}
```

### 7.3 LSTM深度学习模型

**模型架构**:
```python
# train_lstm.py
class MoistureLSTM(nn.Module):
    def __init__(self, input_size=7, hidden_size=64, num_layers=2, output_size=1):
        super(MoistureLSTM, self).__init__()
        self.hidden_size = hidden_size
        self.num_layers = num_layers
        
        self.lstm = nn.LSTM(
            input_size=input_size,
            hidden_size=hidden_size,
            num_layers=num_layers,
            batch_first=True,
            dropout=0.2
        )
        
        self.fc = nn.Sequential(
            nn.Linear(hidden_size, 32),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(32, output_size)
        )
    
    def forward(self, x):
        # LSTM层
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        c0 = torch.zeros(self.num_layers, x.size(0), self.hidden_size)
        out, _ = self.lstm(x, (h0, c0))
        
        # 全连接层
        out = self.fc(out[:, -1, :])
        return out
```

**输入特征**:
- 土壤湿度 (soil_moisture)
- 环境温度 (temperature)
- 环境湿度 (humidity)
- 光照强度 (light_intensity)
- CO2浓度 (co2_concentration)
- 小时 (hour_of_day)
- 日份 (day_of_year)

```java
// MultiObjectiveOptimizer.java
public class MultiObjectiveOptimizer {
    
    public List<IrrigationSolution> optimize(OptimizationContext context) {
        // 1. 初始化种群
        List<IrrigationSolution> population = initializePopulation(100);
        
        // 2. 迭代优化
        for (int gen = 0; gen < 100; gen++) {
            // 非支配排序
            List<List<IrrigationSolution>> fronts = fastNonDominatedSort(population);
            
            // 计算拥挤度
            for (List<IrrigationSolution> front : fronts) {
                calculateCrowdingDistance(front);
            }
            
            // 选择、交叉、变异
            List<IrrigationSolution> offspring = generateOffspring(population);
            
            // 合并并选择下一代
            population = selectNextGeneration(population, offspring);
        }
        
        // 3. 返回帕累托前沿
        return getFirstFront(population);
    }
}
```

```java
public void learnFromResult(Long logId, double actualMoistureAfter) {
    IrrigationLog log = irrigationLogMapper.selectById(logId);
    
    // 1. 计算预测误差
    double predicted = log.getPredictedMoisture();
    double actual = actualMoistureAfter;

    if (error > 5.0) {  // 误差超过5%

        updateAlpha(log.getMonitorPointId(), newAlpha);
```
